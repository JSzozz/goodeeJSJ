속성선택자

태그에 서렁되어 있는 속성값을 기준으로 태그(element)를 선택하는 것
단일태그를 가져올 때 많이 사용함
a, img, vidio, audio, input 등에 많이 사용
표현 : 선택자(생략)[속성명 (연산자) 속성값] 1. = : 동일 비교
2. ^= : 시작하는 문자가 동일한 값을 찾을 때
3. |= : -를 기준으로 문자가 동일한 값을 찾을 때(첫번째 '-' 앞 값만 처리해줌!)
ㄴ('-'뒤에 있는 것 x, '-'앞에 있는 것 o)
4. &= : 끝나는 문자가 동일한 값을 찾을 때
5. *= : 포함된 값을 찾을 때
6. ~= : 띄어쓰기를 기준으로 일치값을 찾을 때

    문자 선택자

    ::first-letter : 문자의 첫 글자에만 style을 적용할 때
