공백을 처리하는 속성

white-space속성을 이용한다 normal : 기본표시 방법 nowrap : 모든 문장을 한 줄로 표시해주는 기능 *길이가 계속 길어지면 지정한 영역을 벗어나버림 pre : pre태그와 동일, 작성된 대로 표시 pre-line : 공백을 하나만 표시, 영역을 벗어나면 개행처리 pre-wrap : 공백을 있는대로 표시, 영역을 벗어나면 개행처리

(pre2) 건물 사이에 피어난 장미 제발 살아남아 줬으면 꺾이지 마 잘 자라줘 온몸을 덮고 있는 가시 얼마나 힘이 들었으면 견뎌내줘서 고마워
(preline) 건물 사이에 피어난 장미 제발 살아남아 줬으면 꺾이지 마 잘 자라줘 온몸을 덮고 있는 가시 얼마나 힘이 들었으면 견뎌내줘서 고마워
